Raw2Event: High-Fidelity Event Camera Simulation

Real-Time Event Generation from Raw Frames.

Raw2Event

Raw2Event is a hardware-software system that converts standard camera frames into neuromorphic event streams in real-time.

The Core: ISP Bypass

Unlike typical simulators, it bypasses the Image Signal Processor (ISP) to access 10-bit raw Bayer data. This preserves the full potential of the sensor, providing Higher Dynamic Range (HDR) and more faithful intensity values than processed RGB streams.

Technical Highlights

  • Embedded Performance: Re-implemented using NumPy-based vectorized operations, achieving 33 FPS on Raspberry Pi 5.
  • Autofocus Support: Leverages conventional sensor features (like PDAF) to provide sharp event boundaries, a feature missing in most commercial event cameras.
  • Physics-Based Model: Built on the DVS-Voltmeter model, using Inverse Gaussian distribution to simulate realistic temporal jitter and noise.

Validation Pipeline

The system was calibrated using a robotic arm to capture perfectly synchronized data between a Raspberry Pi Camera and a professional DAVIS346 event sensor.

User Control

It offers six tunable physical parameters (k1 to k6) to control contrast sensitivity, dark noise, and pixel leakage, allowing users to emulate specific hardware behaviors or lighting conditions.

Technical Overview

  • System Raspberry Pi 5
  • Input RPi Camera Module 3
  • Output 33 FPS Event Stream
  • Core ISP Bypass / NumPy
  • Validation Dobot Arm + DAVIS346
